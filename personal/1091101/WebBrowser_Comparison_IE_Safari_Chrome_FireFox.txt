
IE, Chrome, Safari, FireFox 들의 각각의 장단점들...

1. IE

  내가 처음 인터넷을 사용했던 때가 초등학교 때 였는데, 전화선으로 인터넷을 사용했었던 윈도우 95 이전 시절? 부터 윈도우를 설치하면 그 안에 IE가 있어,
 우리들과 친숙했던 IE였으므로, 그 시절에 넷스게이프라는 브라우저랑 같이 사용하여보았는데, 넷스게이프는 뭔가 내가 사용하기 싫어서 사용하지 않았다.
 일반적으로 사람들이 많이 사용을 함으로 써의 장점이 큰것 같다. 그렇기 때문에 너무 익숙하여, 기본이 되어 버린 것 같다. 엑티브 X 사용이 많아 장점이다.
 단점으로는 다른 브라우저들 보다는 느리다. 불안정 하고, 스펨차단이나 그런 사이트가 막히는 것이 지금은 좀 나아 졌지만, 예전에는 차단이 잘안됬던 것으로 알고 있다.
 옵션에서 보안 설정을 할 수 있는데, 보안이 좀 좋지 않은 것 같다.

2. Chrome

  Chrome을 예전부터 들어와서 알고는 있었지만 IE에 익숙해져서 인지는 모르겠지만 사용하지는 않았다. Chrome을 사용한지는 3개월정도 되는데,
 내가 스마트폰을 바꾸면서이다. 스마트폰으로 인터넷을 할 때 뭔가 인터넷이 부족한 듯 해보이고 불안정 하고, 느리고, 그래서 찾아 낸 것이 Chrome브라우저이다.
  "신개념 브라우저"이런 슬로건을 가지고 나와서 모바일 버전으로 사용을 하고 있는데, 빠르고, 예쁘고 괜찮았다. 그래서 나의 PC에서도 사용 하려 했든데,
 단점으로 뭔가 용량을 차지 해야된다고 해야되나? 하튼 그렇다. 내 PC에서 사용 해보았을 때 그런 느낌을 받았다. 장점으로는 부드럽고, 효과도 첨가 되어 괜찮고,
 파일 다운로드 할때 하단 왼쪽에 떠서 창이 깔끔하고, 서핑할 때 페이지가 넓게 보여 좋다. 괜찮은 브라우저이다. 모바일로 사용하는 것이 좋은 것 같다.

3. Safari

  Safari? 소프트웨어 공학 수업시간에 처음 듣는 브라우저였다. 이런 브라우저가 있나 싶었다. 아직 내가 정보가 느린가 보다하고 좀 더 분발해야 겠다 하면서,
 Safari를 사용해 보았다. 아직 ios를 많이 사용 한적은 없고 친구들 아이폰이나 조금씩 만지면서 신기해 하였는데, Safari는 아이폰이나 아이패드를 만지는 느낌이다.
 장점으로 책갈피, 즐겨찾기 기능에 뭔가 특화가 되어 빠르게 찾는 것이 좋은것 같고, 예쁘고, 한눈에 보기 쉬웠다. 또한 영어 스펠링, 문법을 바로 확인 할 수도 있었다.
 조금 신기 했다. 서핑하는데 속도가 다른 브라우저들과 달리 제일 뭔가 가장 빨랐다. 그러나 Safari 사용을 위해 서핑을 하다보면 IE와는 다르게 Safari버전으로 
 따로 설치 할 것들이 있어 귀찮다.

4. FireFox

  FireFox 언젠가 부터 내가 주로 사용하는 브라우저가 되었다. 그 이유 중 하나의 장점으로는 IE를 주로 사용하는데, IE와 닮았다는 점과 보안도 괜찮고,
 서핑할 때 안정적이고 빠른 편에 속하기 때문이다. 파일을 다운받을 때 다운 받은 목록이 바로 뜨게 되어, 다운 받은 파일 관리가 잘 된다. Chrome과 같이 
 화면이 시원하게 쫙 보이고 좋다. 단점이라고 하면, 즐겨찾기는 바로 볼 수 없는 것이 있다. 그러나 이동 탭에서 전에 보았던 기록들을 다시 볼 수 있어서 좋다.
 아무레도 한국은 IE에 보편화 되고 인터넷 환경도 외국과 달라서 엑티브X 라든지 이런 다른 설치 문제들로 FireFox의 단점이 있겠다.

5. 소감

  이번 과제를 통해 브라우저들을 조금 더 알게 되고 사용을 해보게 된 계기가 되었다. IE 빼고 액티브 X에서 걸림돌이 많은 것같다. 모두의 장단점 들이 있겠지만,
 사용자가 잘 사용하는 것 좋아하는 브라우저를 사용하면 좋을 것 같다. 우리 나라 인터넷 환경도 바뀌어 IE 말고 다른 브라우저의 활성화도 기대해 본다.
