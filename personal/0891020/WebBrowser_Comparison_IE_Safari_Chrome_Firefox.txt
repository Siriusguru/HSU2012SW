IE
9버전으로 업그레이드되면서 디자인적인 면에서 단순해 졌는데 크롬과 비슷한 느낌을 받았다.
장점 
학교, 집, 관공서 등에서 가장 많이 쓰는 웹 브라우저 답게 친숙해서, 조작이 쉽다.
인터넷 익스플로러9으로 바뀌면서 탭 분리기능이 나타나면서 탭을 분리하여 자료 비교 등을 하기에 편리하다
북마크 사용, 추가가 간편하다
단점
버전 업그레이드로 빠른 속도를 장점으로 뽑았지만 아직까진 타 웹브라우저에 비해 속도를 장점으로 뽑을 순 없다.
그리고 9버전은 윈도우 비스타와 윈도우7만 지원하여서 XP사용자들은 사용할 수가 없다.
장점이자 단점일수도 있는 점인데 액티브X 이다. 설치함으로써 특정 데이터를 볼 수 있게 하지만 보안성의 문제가 있다.

크롬
처음 셋팅 된 그대로 열었을 때 새 탭 기능이 어색하지만, 구글, YouTube, Gmail을 하는 사용자라면 편리할 것 같다.
장점
동기화 기능덕분에 크롬에 계정 등록을 해 놓았다면 다른 컴퓨터에서도 설정과, 북마크 기능을 사용할 수 있다.
속도가 빠르다. 최고 사양의 컴퓨터가 아니라면 크롬의 반응속도가 더 빠르다고 느낄 것이다.
단점
IE9에 익숙해서 홈 버튼을 사용하려 할 때 홈 버튼이 없어서 불편하다.
우리나라는 웹페이지는 ActiveX를 사용하는 홈페이지들이 많기 때문에 ActiveX기반 웹페이지를 사용하려면 IE TAB이라는 크롬확장프로그램을 설정해야한다.

파이어 폭스
처음 실행했을 때, 예전 버전의 IE느낌이다. 많은 도구모음들이 있고, 설정을 잘못 했는지 다음 툴 바도 설치되어있다. 도구모음들과 파일메뉴까지 있다 보니 다른 웹브라우저에 비하여 심플한 느낌을 받진 못했다.
장점 
파이어 폭스의 다양한 부가(확장)기능을 설정하면 인터넷을 좀 더 편리하게 사용할 수 있다.
웹브라우저에서 그림 저장할 때 바탕화면으로 드래그하면 저장이 된다.(크롬도 가능하긴 한데 파이어 폭스의 저장 속도가 더 빠르다)
단점
마우스 휠을 사용하여 스크롤을 움직일 때 스크롤의 움직임이 느리다.
페이지 내 미디어항목 때문에 플러그인 설치가 많다. 크롬과 마찬가지로 IE Tab을 다운받아 설정해야 한다.

사파리
타 웹브라우저에 비하여 Top Site 기능이 눈에 들어온다. 
장점
책갈피기능을 사용하기 편하고, 오늘 방문한 페이지도 같이 보여주기 때문에 필요할 때 유용한 것 같다.
TopSite기능을 사용하여 자주 사용하는 페이지를 볼 수 있고, 해당페이지로 이동이 쉽다. 
단점
웹페이지의 탭 분리기능 사용은 가능하나 탭 합치는 기능은 포함하지 않는 것 같다.
탭 기능이 있긴 하지만 새로운 웹페이지를 탭으로 실행 하는 것이 아니라 새 웹브라우저로 실행해서 검색 시 웹브라우저 창만 여러 개가 되어버려 불편하다.
페이지 로딩속도가 크롬, 파이어 폭스에 비하여 느리다.
