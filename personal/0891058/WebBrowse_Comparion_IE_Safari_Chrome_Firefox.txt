파이어폭스
웹브라우저가 돌아 갈 때 IE 와 크롬 과 다르게 프로세서가 
브라우저와 플러그인 두개로 구성되어 단조롭다.
부가기능이 많아서 자신에 스타일에 맞게 꾸밀수 있다.
이 기능이 상당히 좋아서 크롬app 과 알툴바가 따라한거 같다.
하지만 부가기능이 좋다고 많이 설치하면 플러그인 메모리가 증가하므로 
적당히 필요한 만큼 설치하는 절제 정신이 필요하다.
컴퓨터 최적화 과정인 램디스크 설정하기가 약간 번거롭다. 
처음 브라우저 ui 가 깔끔하다 하지만 처음 사용할 때 필요한 기능이 어디있는지 잘 모를수 있다.
비영리로 제작된거라 신뢰가 되고, 사생활모드, 보안 등 상당히 신경을 썼다. 
이번에 브라우저를 다 써보면서 파이어폭스를 처음 쓰는데 가장 마음에 드는 브라우저인것 같다.
webwait 사이트에서 다른브라우저와 속도 측정해보았는데 속도 차이는 미미해서 실사용에서는 차이를 못 느끼기 때문에 
속도보다는 에러나 프리징이 없는 안정성이 더 끌리게 되었다. 
 

크롬 
브라우저가 상당히 무겁다 메모리 점유율과 cpu 점유율이 높지만 따로 설정 안해도 엄척나게 빠르다. 
하지만 플래시 재생에 제한이 있다. 또 국내 웹 환경에서 안 열리는 웹페이지 창이 많다.
가끔 IE에서 자주 발생하는 프리징이 크롬에도 있다. IE에서는 옵션에서 GPU 렌더링 대신
소프트웨어 렌더링 사용을 하면 해결되지만 크롬은 해결방법을 모르겠다. 그래서 사용하기 꺼리게 된다.
구글에서 만든 웹이라 구글에서 만든 여러 프로그램과 호환성이 좋다. 특히 자동 구글 변역기능으로 외국 사이트 즐길 수 있다.
크롬 app 으로 부분적으로 알툴바 기능 어느정도 가능하다 그리고 IE 미니 탭 기능을 설치하면 IE 웹페이지를 볼 수 있다.
그래서 크롬에서 확인 할 수 없는 웹페이지를 볼 수 있다.
구글 아이디 있으면 어느 컴퓨터라도 북마크를 자신이 사용하던데로 사용가능하다.
브라우저 설정이 편리하다 GPU 가속 과 램디스크 사용으로 날라다니는 웹속도 자랑할수 있다.

IE 
기존 사용하던 습관으로 상당히 편리하게 사용이 가능하다 하지만 사용하던 습관으로 다른 브라우저 사용을 극히 꺼리게 된다.
알툴바 설치로 대부분의 부가기능 가능하다. 다른 브라우저는 하나 하나 개별 설치와 다르게 너무 편리하다.
대부분의 사람들은 윈도우 설치후 자연스럽게 IE 사용을 당연시 하면서 브라우저의 다양성을 알지 못하고 우물안 개구리 신세된다.
큰 단점으로 느린 속도와 컴퓨터 자원을 많이 사용 그리고 보안성이 떨어지며 각종 오류로 불안정하다.
하지만 윈도우7 사용자는 IE9를 사용할수 있는데 새로 업그레이드 되면서 구버전 사용자들은 생소하고 불편했지만 
다른 브라우저들과 비슷한 디자인이 되어서 접근성이 쉽게 되었다. 또 속도와 안정성이 많이 개선이 되었다.
그리고 엑티브 엑스 환경으로 대한민국 사람들은 꼭 사용해야 은행과 같은 웹페이지에 접근할 수 있다.
하지만 막무간에 엑티브 엑스 설치로 컴퓨터가 느려질 위험이 있다. 
최적화를 위해 램디스크 설정시 대용량 파일 다운로드 설치할 때 램에 임시파일 저장이 되서 램디스크 용량 초과시 설치 불과로 불편하다.

사파리
오직 mac os 사용자를 위해 맞춤 설계되고 애플 감성 디자인을 느낄 수 있다. 
또한 트랙패드로 편하게 사용이 가능하다고 하는데 내 노트북은 맥이 아니라 사용을 못해봤다.
개인적인 사용 패턴은 노트북 터치패드 제스처를 이용해서 마우스 없이 사용하는데 다른 브라우저와 다르게 제스처를 인식을 못해서 불편했다.
또, 요즘 많이 사용되는 마우스 제스처 기능 같은 부가 가능을 확장 프로그램으로 가능하다.
하지만 다른 브라우저와 조금 떨어지는 감이 없지 않아 있다. 보통 오른쪽 마우스 클릭이 액션 제스처인데
가운데 마우스 버튼을 기준으로 만들어서 설정으로 오른쪽 버튼으로 바꾸면 브라우저 메뉴창이 떠서 사용이 잘 안된다.
사파리 기능으로 읽기 목록이 있는데 다른 브라우저로 보통 북마크 기능을 사용해서 그런지 아직 잘 사용을 못하겠다.
아이클라우드 기능 활용하면 자신이 즐겨 가던 북마크를 다른 환경에서 사파리 사용시 똑같이 이용할수 있다.
브라우저 설정과정에서 유일하게 램디스크 설정 못했다. 주로 맥 환경에 마쳐서 설명이 되있어서 윈도우 환경 설명이 부실하고
어렵게 되어서 포기했다.
크롬, IE, 파이어폭스, 사파리 4개의 브라우저를 네이버로 띄었을 때 프로세서 용량이 제일 작았고 
cpu 사용량 적은 것으로 보아서 제일 가벼운 브라우저 같다.
